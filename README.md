# Visualizer

This repository contains:
- **Neural Network Visualizer** - Interactive deep learning model explorer (root `/index.html`)
- **Model Accuracy Presentation** - Dark mode presentation comparing ML models (`/presentation/index.html`)

## GitHub Pages Setup

To host this on GitHub Pages:

1. Go to your repository **Settings** â†’ **Pages**
2. Under **Source**, select **Deploy from a branch**
3. Select **master** (or **main**) branch and **/ (root)** folder
4. Click **Save**

Your site will be available at:
- `https://ashergrayne.github.io/Visualizer/` - Neural Network Visualizer
- `https://ashergrayne.github.io/Visualizer/presentation/` - Model Accuracy Presentation

## Local Development

To run the Neural Network Visualizer locally:
```bash
npm install
npm run dev
```

To view the presentation locally, simply open `presentation/index.html` in your browser.
